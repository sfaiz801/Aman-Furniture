# National Furniture & Electronics

Premium furniture showroom website built with Next.js, Bootstrap 5, and SCSS.

## Tech Stack
- Next.js 14 (Pages Router)
- React 18
- JavaScript (No TypeScript)
- SCSS / Sass
- Bootstrap 5
- Framer Motion
- Swiper.js
- AOS (Animate On Scroll)
- React Icons

## Getting Started

### Installation
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000)

### Build for Production
```bash
npm run build
npm start
```

## Folder Structure
```
src/
├── assets/         # Images, icons, logo files
├── components/     # Reusable UI components
├── sections/       # Page sections
├── data/           # Editable data files
├── styles/         # SCSS architecture
├── utils/          # Helper functions
└── pages/          # Next.js pages
```

## Customization Guide

### Change Colors
Edit `src/styles/abstracts/_variables.scss`

### Update Offers
Edit `src/data/offers.js`

### Update Categories
Edit `src/data/categories.js`

### Update Testimonials
Edit `src/data/testimonials.js`

### Update Business Info
Edit `src/data/company.js`

## Responsive Breakpoints
- Mobile: < 576px
- Tablet: 576px - 991px
- Desktop: > 992px
