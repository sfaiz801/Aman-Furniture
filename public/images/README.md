# Image Assets

Place your actual images in this folder. Use the following naming convention:

## Required Images:
- `hero-bg.jpg` - Hero section background (1920x1080)
- `offer1.jpg`, `offer2.jpg`, `offer3.jpg` - Offer card images (800x500)
- `cat-bed.jpg`, `cat-sofa.jpg`, `cat-almari.jpg`, `cat-dressing.jpg`, `cat-electronics.jpg`, `cat-chairs.jpg`, `cat-tables.jpg`, `cat-wedding.jpg` - Category images (600x400)
- `factory-main.jpg` - Factory showcase image (1200x800)
- `user1.jpg` - `user5.jpg` - Testimonial avatars (200x200)

## Gallery Images:
Place in `gallery/` subfolder:
- `bed1.jpg`, `bed2.jpg` - Bed photos
- `sofa1.jpg`, `sofa2.jpg` - Sofa photos
- `almari1.jpg` - Almari photos
- `wedding1.jpg`, `wedding2.jpg` - Wedding setup photos
- `electronics1.jpg` - Electronics photos
- `factory1.jpg`, `factory2.jpg` - Factory photos
- `dressing1.jpg` - Dressing table photos
- `chair1.jpg` - Chair photos

## Optimization Tips:
- Use WebP format for better performance
- Compress images before adding (use TinyPNG)
- Keep hero image under 200KB
- Gallery images under 100KB each
- Use lazy loading (already implemented in code)
